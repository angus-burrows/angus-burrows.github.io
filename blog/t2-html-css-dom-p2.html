Technical II

What is grid based design?

This is a technique used in design to aid in placing content onto a page. It is achieved by forming a grid that sets rows and columns for the content to go into. Grids helps establish spacing and hierarchy.

Whats all the hype about responsive webpages?

The hype is around making the internet accessible from every device. The term responsive means that you can create rules that tell the webpage to change based on what size screen you are using.

What is semantic structure?

Semantic structure describes the language used when using tags and elements in HTML and CSS. A semantic tag is able to tell the user what it does without giving detailed description, since it's universal.
